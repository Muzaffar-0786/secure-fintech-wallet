# 💳 Secure FinTech Wallet API (MVP)

A modern, secure and scalable FinTech Wallet API built with **FastAPI**. This project demonstrates a complete digital wallet workflow including authentication, wallet management, transactions and reward system.

Designed as a **Minimum Viable Product (MVP)** with a clean architecture that can be extended into a production-ready fintech application.

---

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 💼 Digital Wallet
- 💰 Balance Management
- ➕ Add Money (Demo)
- ➖ Spend Money
- 📜 Transaction History
- 🎁 Daily Reward System
- 📊 Reward History
- 📄 Interactive Swagger API Documentation
- ⚡ FastAPI High Performance Backend
- 🗄️ PostgreSQL Database
- 🛡️ Password Hashing with bcrypt

---
